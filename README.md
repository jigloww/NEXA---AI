# NEXA AI

NEXA AI adalah Personal Multi-Agent Assistant yang dirancang untuk membantu pengguna dalam bidang akademik, produktivitas, bisnis, dan investasi.

## Features

* Academic Assistant
* Productivity Assistant
* Business Assistant
* Investment Assistant
* Long-Term Memory
* Telegram Integration
* Multi-Agent Architecture

## Planned Agent Structure

```text
NEXA AI
│
├── Master Agent
├── Memory Agent
├── Productivity Agent
├── Academic Agent
├── Business Agent
└── Investment Agent
```

## Technology Stack

* Python
* FastAPI
* PostgreSQL
* ChromaDB
* Telegram Bot API
* LangGraph
* OpenAI / Gemini
